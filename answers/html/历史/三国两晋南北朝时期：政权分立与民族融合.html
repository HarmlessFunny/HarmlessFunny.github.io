三国两晋南北朝时期：政权分立与民族融合

三国时期

| 战役 | 交战双方 | 意义 | | -------- | -------------- | ---------------------------- | | 官渡之战 | 曹操、袁绍 | 为曹操统一北方奠定了基础 | | 赤壁之战 | 曹操、孙刘联军 | 为三国鼎立局面形成奠定了基础 |

西晋

  • 建立者：司马炎
  • 建立时间：280年
  • 八王之乱影响：形成我国历史上第一次大规模人口迁徙高潮
  • 北方少数民族内迁
    • 民族：匈奴、羯、鲜卑、羌、氐
    • 影响：形成五胡十六国局面，促进民族交融

江南地区开发

  • 原因：北人南迁，提供劳动力和先进的生产工具和技术；江南地区自然条件优越；南方战乱较少，社会安定；劳动人民的辛勤劳动；统治者的重视
  • 影响：为日后中国古代经济重心的南移奠定了基础

北魏孝文帝改革

  • 措施：迁都洛阳；说汉语，穿汉服，改汉姓，通汉婚
  • 影响：促进了民族交融，增强北魏的实力

科技和文化

  • 祖冲之：计算圆周率至小数点后七位，制定《大明历》

  • 贾思勰：《齐民要术》，我国现存最早的第一部完整的农业科学著作

  • 郦道元：《水经注》

  • 王羲之：《兰亭集序》，笔势“飘若浮云，矫若惊龙”，被称为“书圣”
  • 顾恺之：《女史箴图》，《洛神赋图》
  • 石窟：山西大同云冈石窟，河南洛阳龙门石窟
