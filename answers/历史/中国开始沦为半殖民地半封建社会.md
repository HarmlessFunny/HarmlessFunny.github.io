## 中国开始沦为半殖民地半封建社会

#### 过程

- 鸦片战争：开始沦为半殖民地半封建社会，中国近代史的开端
- 第二次鸦片战争：半殖民地化程度进一步加深
- 甲午中日战争：半殖民地化程度大大加深
- 八国联军侵华战争：完全沦为半殖民地半封建社会

#### 不平等条约

<ul>
  <li>《南京条约》
    <ul>
      <li>割香港岛给英国</li>
      <li>开放广州、福州、厦门、宁波、上海为通商口岸</li>
      <li>英商进出口货物应纳税款，必须经过双方协议</li>
    </ul>
  </li>
  <li>《马关条约》
    <ul>
      <li>割辽东半岛、澎湖列岛、台湾全岛及所有附属各岛屿给日本</li>
      <li>开放沙市、重庆、苏州、杭州为商埠</li>
      <li>赔款2亿两白银</li>
      <li>允许日本在通商口岸开设工厂</li>
    </ul>
  </li>
  <li>《辛丑条约》
    <ul>
      <li>赔款4.5亿两白银</li>
      <li>严禁人民参与反帝活动</li>
      <li>拆毁大沽炮台</li>
      <li>划定北京东交民巷为使馆界</li>
      <li>改总理衙门为外务部</li>
    </ul>
  </li>
</ul>
