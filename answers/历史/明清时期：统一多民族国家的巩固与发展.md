## 明清时期：统一多民族国家的巩固与发展

#### 明

<ul>
  <li>建立时间：1368年</li>
  <li>建立者：朱元璋</li>
  <li>都城：南京</li>
  <li>强化皇权的措施
    <ul>
      <li>废丞相，设立三司</li>
      <li>八股取士</li>
      <li>设锦衣卫、东厂</li>
    </ul>
  </li>
  <li>经济发展
    <ul>
      <li>农业：引进玉米、甘薯、马铃薯、花生、向日葵</li>
      <li>手工业：景德镇青花瓷</li>
      <li>商业：商帮</li>
    </ul>
  </li>
  <li>对外关系
    <ul>
      <li>郑和下西洋，最远到达非洲东海岸和红海沿岸</li>
      <li>戚继光抗倭</li>
      <li>葡萄牙攫取在澳门的居住权</li>
    </ul>
  </li>
  <li>科技文化
    <ul>
      <li>医学：李时珍《本草纲目》</li>
      <li>农业学：徐光启《农政全书》</li>
      <li>宋应星：《天工开物》，中国17世纪的工艺百科全书</li>
      <li>小说：《三国演义》《西游记》《水浒传》</li>
      <li>戏剧：汤显祖《牡丹亭》</li>
    </ul>
  </li>
</ul>

#### 清

<ul>
  <li>建立时间：1636年</li>
  <li>建立者：皇太极</li>
  <li>对边疆的管理
    <ul>
      <li>台湾：设台湾府，建台湾省</li>
      <li>西藏：册封达赖、班禅</li>
      <li>新疆：设伊犁将军</li>
    </ul>
  </li>
  <li>闭关锁国原因：清朝统治认为天朝物产丰富，无所不有，不需要同外国进行经济交流；西方殖民者正向东方扩展势力，清朝统治者担心国家领土主权受到侵犯；惧怕沿海人民同外国人交往，会危及自己的统治</li>
  <li>文学艺术
    <ul>
      <li>小说：《红楼梦》</li>
      <li>昆曲：《牡丹亭》</li>
    </ul>
  </li>
</ul>
